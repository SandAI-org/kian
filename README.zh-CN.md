## 核心特性

1. 多智能体协作：不同模型、不同角色配置可以在同一套工作流中协同完成任务。
2. 开箱即用的文档管理：记录想法、加速写作，并为你最强的智能体沉淀知识库。
3. 定时任务：自动执行周期性工作与日常提醒。
4. 后台长任务：把耗时工作交给智能体，并让它在后台持续运行。
5. 内置应用开发能力：让软件创建更易上手，随时构建小工具和小游戏。
